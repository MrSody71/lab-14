Создай CI на GitHub Actions: три job (go/python/rust) + один gate-job на
prompts. Никаких других изменений в репозитории, только файлы в
.github/workflows/.

=== .github/workflows/ci.yml ===

name: CI

on:
  push:
    branches: [main]
  pull_request:

jobs:
  go:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-go@v5
        with:
          go-version: "1.22"
          cache: true
      - name: Sync workspace
        run: go work sync
      - name: Build
        run: |
          go build ./go-collector/...
          go build ./arrow-server/...
          go build ./mock-owm/...
      - name: Test
        run: |
          go test ./go-collector/... -race -count=1
          go test ./arrow-server/... -race -count=1
          go test ./mock-owm/... -race -count=1
      - name: Lint
        uses: golangci/golangci-lint-action@v6
        with:
          version: v1.61.0
          args: ./go-collector/... ./arrow-server/... ./mock-owm/...

  python:
    runs-on: ubuntu-latest
    strategy:
      fail-fast: false
      matrix:
        project: [py-analyzer, py-asyncio-collector, dashboard]
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: "3.11"
      - uses: astral-sh/setup-uv@v3
        with:
          enable-cache: true
      - name: Sync
        working-directory: ${{ matrix.project }}
        run: uv sync
      - name: Lint
        working-directory: ${{ matrix.project }}
        run: uv run ruff check
      - name: Test
        working-directory: ${{ matrix.project }}
        run: uv run pytest -v

  rust:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: dtolnay/rust-toolchain@stable
        with:
          components: rustfmt, clippy
      - uses: Swatinem/rust-cache@v2
        with:
          workspaces: rust-validator
      - name: Format
        working-directory: rust-validator
        run: cargo fmt --check
      - name: Clippy
        working-directory: rust-validator
        run: cargo clippy --all-targets -- -D warnings
      - name: Test
        working-directory: rust-validator
        run: cargo test
      - name: Build release
        working-directory: rust-validator
        run: cargo build --release

  prompts-gate:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Check prompts/ exists and has files
        run: |
          if [ ! -d prompts ]; then
            echo "::error::prompts/ directory missing"
            exit 1
          fi
          count=$(find prompts -name "*.md" -type f | wc -l)
          if [ "$count" -lt 1 ]; then
            echo "::error::No prompt files in prompts/"
            exit 1
          fi
          echo "Found $count prompt files."
      - name: Scan for leaked secrets
        run: |
          if grep -r -E "(OWM_API_KEY|sk-[a-zA-Z0-9]{20,}|ghp_[a-zA-Z0-9]{30,})" prompts/; then
            echo "::error::Possible secret leaked in prompts/"
            exit 1
          fi
          echo "No secrets found."

=== .github/workflows/codeql.yml ===

name: CodeQL

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]
  schedule:
    - cron: "0 6 * * 1"

jobs:
  analyze:
    runs-on: ubuntu-latest
    permissions:
      actions: read
      contents: read
      security-events: write
    strategy:
      fail-fast: false
      matrix:
        language: [go, python]
    steps:
      - uses: actions/checkout@v4
      - uses: github/codeql-action/init@v3
        with:
          languages: ${{ matrix.language }}
      - uses: github/codeql-action/autobuild@v3
      - uses: github/codeql-action/analyze@v3

ПРОВЕРКА:
Запусти локально валидацию YAML:
  python3 -c "import yaml; yaml.safe_load(open('.github/workflows/ci.yml')); yaml.safe_load(open('.github/workflows/codeql.yml'))"

Должно отработать без ошибок. Если python3-yaml нет — используй любой
другой YAML-парсер (yq, например) или скажи мне.
